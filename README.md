# ZeroJS-Drive 📂

Легковесный, кроссплатформенный локальный HTTP-сервер для удобной навигации по файловой системе и обмена файлами внутри домашней сети. Построен на базе C# и ASP.NET Core Razor Pages без единой строки JavaScript.

## 🛠 Архитектурные особенности

* **Strict Zero-JS**: Никакого фронтенд-оверхеда. Весь рендеринг происходит строго на стороне сервера (SSR).
* **POST-Redirect-GET (PRG)**: Навигация по подпапкам защищена от повторной отправки форм и корректно сохраняет историю переходов в браузере.
* **Сетевой интерфейс 0.0.0.0**: Сервер слушает все доступные сетевые интерфейсы машины, обеспечивая мгновенный доступ к файлам с телефона или соседних ПК в локальной сети.
* **Адаптивная сетка**: Интерфейс построен на чиском CSS Grid и Flexbox. Идеально отображается как на мониторах, так и на экранах смартфонов.

## 🚀 Быстрый старт (Запуск без установки .NET)

Проект скомпилирован в виде независимых исполняемых файлов и не требует наличия установленной среды .NET на целевой машине.

### Вариант 1: Запуск на Windows
1. Скачайте архив с релизом для Windows (`win-x64`).
2. Запустите исполняемый файл `ZeroJS-Drive.exe`.

### Вариант 2: Запуск на Linux
1. Скачайте релиз для Linux (`linux-x64`).
2. Откройте терминал в папке с файлом, выдайте права на исполнение и запустите его:
   ```bash
   chmod +x ZeroJS-Drive
   ./ZeroJS-Drive
   ```

## 📱 Как подключиться с телефона или другого ПК

По умолчанию сервер принимает подключения на порту **5286**.

1. Узнайте локальный IP-адрес компьютера, на котором запущен сервер:
   * **На Windows**: выполните команду `ipconfig` в командной строке.
   * **На Linux**: выполните команду `ip a` или `hostname -I` в терминале.
2. Найдите ваш IPv4-адрес в локальной сети (обычно начинается на `192.168.x.x` или `10.x.x.x`).
3. На смартфоне или другом устройстве, подключенном к той же Wi-Fi сети, откройте браузер и перейдите по адресу:
   ```text
   http://<ВАШ_IP_АДРЕС>:5286
   ```
   *Пример: http://192.168.1.45:5286*
