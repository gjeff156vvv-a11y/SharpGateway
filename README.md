# ZeroJS-Drive 📂

Легковесный, кроссплатформенный локальный HTTP-сервер для удобной навигации по файловой системе и обмена файлами внутри домашней сети. Построен на базе C# и ASP.NET Core Razor Pages без единой строки JavaScript.

## 🛠 Архитектурные особенности

* **Strict Zero-JS**: Никакого фронтенд-оверхеда. Весь рендеринг происходит строго на стороне сервера (SSR).
* **POST-Redirect-GET (PRG)**: Навигация по подпапкам защищена от повторной отправки форм и корректно сохраняет историю переходов в браузере.
* **Сетевой интерфейс 0.0.0.0**: Сервер слушает все доступные сетевые интерфейсы машины, обеспечивая мгновенный доступ к файлам с телефона или соседних ПК в локальной сети.
* **Адаптивная сетка**: Интерфейс построен на чиском CSS Grid и Flexbox. Идеально отображается как на мониторах, так и на экранах смартфонов.

## 🚀 Быстрый старт (Запуск готовых релизов)

Проект полностью автономен, скомпилирован в независимые бинарники со встроенным рантаймом и не требует установки .NET SDK на целевой машине.

### Вариант 1: Запуск на Windows
1. Перейдите в раздел **Releases** на GitHub и скачайте архив `windows_release.zip`.
2. Распакуйте архив в любую удобную директорию.
3. Запустите исполняемый файл `HttpFileSystem.exe` двойным кликом.

### Вариант 2: Запуск на Linux
1. Скачайте архив `linux_release.tar.gz` из релизов.
2. Распакуйте его через терминал:
   ```bash
   tar -xzvf linux_release.tar.gz -C ./ZeroJS-Drive
   cd ./ZeroJS-Drive
   ```
3. Выдайте права на исполнение бинарнику и запустите сервер:
   ```bash
   chmod +x HttpFileSystem
   ./HttpFileSystem
   ```

## 📱 Локальная сеть и порты

По умолчанию сервер начинает слушать порт **5286** на всех сетевых интерфейсах машины (`0.0.0.0`). 

Для подключения со смартфона или другого ПК, находящегося в той же Wi-Fi сети:
1. Узнайте ваш локальный IP-адрес (команда `ipconfig` в Windows или `hostname -I` в Linux).
2. Откройте браузер на мобильном устройстве и введите адрес вида: `http://<ВАШ_IP>:5286` (например, `http://192.168.1.45:5286`).
